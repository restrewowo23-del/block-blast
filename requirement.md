프로젝트명: 블록 퍼스트(Block Burst) - 웹 기반 퍼즐 게임
1. 목적
사용자 경험: 드래그 앤 드롭 방식을 통해 직관적이고 중독성 있는 퍼즐 맞추기 경험 제공.

학습 목표: HTML DOM 조작을 통한 동적 그리드 생성, JavaScript 이벤트 리스너(Drag & Drop) 활용, 배열 기반의 격자 로직 구현.

2. 필수 기능
보드 시스템: <div> 태그를 활용한 10x10 격자 보드 생성 및 관리.

블록 생성: 매 라운드마다 무작위 형태(I, L, O, T 등)와 색상의 블록 3개 하단 생성.

배치 로직: * HTML5 Drag and Drop API 또는 Mouse Event를 이용한 블록 배치.

보드 내 빈 칸 확인 후 배치 가능 여부 시각적 표시.

배치 후 하단 블록 3개를 모두 사용하면 새로운 블록 3개 생성.

라인 제거: 가로 또는 세로 10칸이 가득 차면 즉시 제거 및 보너스 점수 부여.

점수 시스템: * 블록 배치 시 구성 타일 개수만큼 점수 획득.

라인 제거 시 가산점(멀티 라인 제거 시 콤보 점수) 적용.

게임 오버: 보드 내에 현재 남은 블록 중 어떤 것도 배치할 수 없는 상태가 되면 게임 종료 팝업 출력.

3. 디자인 요구 사항
DOM 기반 스타일링: CSS Flexbox/Grid를 사용하여 10x10 보드를 정교하게 배치.

시각적 피드백: * 드래그 중인 블록이 놓일 위치를 보드 위에 미리 보기(Semi-transparent).

라인 제거 시 CSS Animation/Transition을 활용한 반짝임 또는 페이드 효과.

인터페이스: 상단에 현재 점수(Score)와 최고 기록(Best) 표시, 중앙에 게임 보드, 하단에 블록 생성 구역 배치.

4. 기술 스택 및 환경
언어: HTML5, CSS3, JavaScript (Vanilla JS).

그래픽 구현: DOM 방식 (document.createElement 및 클래스 제어).

개발 환경: VS Code (Visual Studio Code).

로컬 서버: * VS Code Live Server 확장 프로그램을 사용하여 실시간 코드 반영.

브라우저는 최신 버전의 Chrome 또는 Edge 권장.
